## Node

*   Install latest node version
    *   nvm install node
*   Use latest node version
    *   nvm use node
*   List all node versions
    *   nvm list
*   Link to NVM doc
    *   [https://github.com/nvm-sh/nvm](https://github.com/nvm-sh/nvm)
